# Carpeta para imágenes de "Mejor Jugadora de la Temporada"

Coloca aquí las fotos de las jugadoras ganadoras del premio de temporada.

## Nomenclatura:
- Nombrar con el `id_premio` de la tabla `jugadora_del_mes`
- Ejemplo: `1.jpg`, `2.png`, `5.webp`

## Especificaciones:
- Dimensiones: 600x800px (3:4 ratio)
- Formatos: .jpg, .png, .webp
- Peso máximo: 500 KB

Ver `GUIA_IMAGENES_PREMIOS.md` en la raíz del proyecto para más detalles.
